

```
清除容器

清除镜像
docker rmi -f $(docker images -q) 
```



> linux可以ping通docker容器
>
> * 每启动一个docker容器，docker就会分配一个IP，桥接模式，使用的技术是evth-pair技术
> * 每启动一个docker，docker就会多一个网卡
> * evth-pair就是一对虚拟设备接口，都是成对出现的，一段连着协议，一段彼此相连
> * 正因为有这个特性evth-pair充当一个桥梁，连接各种虚拟网络设备
